# Student Study Planner Web App

A web-based study planner to help students manage subjects, track exam dates, generate study schedules, and maintain daily task checklists.

## Features
- Add subjects with exam dates
- Set subject priority (High/Medium/Low)
- Auto timetable generation
- Daily task checklist
- LocalStorage support for saving data

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Run
Open the `index.html` file in any modern web browser.
